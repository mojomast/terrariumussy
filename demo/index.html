How it works

Install

Add terrarium to your Python project with pip. Zero sibling packages required — runs in stub mode.

pip install terrarium

Run

Point terrarium at your codebase. It scans files, reads git history, and loads adapter data.

terrarium health ./my-project

Understand

Read the ecosystem. See which modules are thriving, stressed, infected, or decaying.

terrarium watch ./my-project
